<h2>YOLO Image Detection App (Streamlit)</h2>

<p>This project is a simple object detection web app built using YOLO (Ultralytics) and Streamlit.</p>
<p>Users can upload an image, and the app runs a trained YOLO model (best.pt) to detect objects and display the annotated prediction output.</p>

<h4>Features</h4>
- Upload images (jpg, jpeg, png)
- Object detection using a custom YOLO model
- Displays both the original and predicted image
- Simple and interactive Streamlit UI

<h4>Technologies Used</h4>
- Python
- Ultralytics YOLO
- Streamlit
- PIL (Image Processing)
